Apa Itu HTML?

Pernahkah kamu membuka sebuah website dan bertanya-tanya bagaimana tampilan dan isi di dalamnya bisa tersusun rapi? Nah, semua itu berawal dari HTML! Yuk, kita cari tahu bersama apa itu HTML dan apa saja yang perlu kita ketahui tentang bahasa pemrograman yang satu ini.

Pengertian

HTML (Hyper Text Markup Language) adalah bahasa standar yang digunakan untuk membuat dan menyusun halaman web. HTML berfungsi untuk menentukan struktur, isi, dan elemen-elemen pada halaman web seperti teks, gambar, tautan, tabel, dan lainnya. HTML menggunakan tag-tag tertentu untuk menandai setiap bagian dari dokumen sehingga browser dapat menampilkannya dengan benar.

Versi pertama HTML dirilis oleh Tim Berners Lee pada tahun 1991, yang memiliki 18 tag. Sejak saat itu, setiap kali ada versi barunya, pasti akan selalu ada tag dan attribute (tag modifier) yang juga baru.

HTML adalah fondasi dari semua halaman web. Tanpa HTML, tidak ada struktur yang bisa diatur untuk menampilkan konten pada web. HTML memungkinkan kita untuk membuat teks, gambar, video, form, tabel, dan banyak elemen lainnya yang diperlukan untuk membangun website berfungsi penuh.


Struktur HTML

Struktur dasar HTML membentuk fondasi dari setiap halaman web. Dengan memahami elemen-elemen inti dan cara menyusunnya, kamu akan dapat membuat halaman web yang terorganisir dan fungsional.

Berikut adalah struktur dasar HTML :

  1. < !DOCTYPE html > : Ini memberitahu browser bahwa dokumen ini menggunakan HTML.

  2. < html > : Elemen root dari dokumen HTML.

  3. < head >: Bagian ini berisi informasi meta tentang dokumen, seperti judul halaman.

  4. < title >: Judul halaman yang ditampilkan di tab browser.

  5. < body >: Bagian ini berisi konten utama halaman web, seperti teks, gambar, dan elemen lainnya.


Tags HTML

Elemen teks dalam HTML digunakan untuk menyusun dan menampilkan konten teks di halaman web. Elemen-elemen ini memungkinkan kamu untuk mengatur teks dengan berbagai cara, yakni menambahkan judul, paragraf, daftar, kutipan, dan banyak lagi.

Berikut adalah beberapa elemen tags yang umum digunakan dalam HTML:

  • < h1 > sampai < h6 >: Digunakan untuk membuat heading atau judul dengan tingkat kepentingan yang berbeda.

  • < p >: Digunakan untuk membuat paragraf teks.

  • < a >: Digunakan untuk membuat tautan (link) ke halaman lain atau sumber daya lainnya.

  • < img >: Digunakan untuk menyisipkan gambar ke dalam halaman web.

  • < ul >, < ol >, dan < li >: Digunakan untuk membuat daftar tak berurutan (unordered list) dan daftar berurutan (ordered list).

  • < div >: Digunakan untuk mengelompokkan elemen-elemen HTML bersama-sama, sering digunakan untuk tujuan styling dengan CSS.
